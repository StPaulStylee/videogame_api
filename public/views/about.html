BrokenClock! is a CRUD (Create, Read, Update, and Delete) application created by JavaScript Programmer and Web Developer, Jeffrey Miller. The creation of this application was made possible in large part due to the video game API created by Giant Bomb. Big thanks to those guys. If you like what you see and what to discuss how it was made or potential opportunities, please feel free to email me or connect via LinkedIn.


The API created by Giant Bomb allows for much more content than just video game titles. Information on development firms, publishers, characters, videogame swag, etc. can all be found via the API. In the future BrokenClock! will take advantage of this in depth search capability. Also in the future this application will allow for user authentication, making the experience personalized for each individual.
